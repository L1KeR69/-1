# Портал дистанционного обучения

## Назначение проекта
Портал дистанционного обучения — это Open Source LMS-решение для онлайн-школ и корпоративных университетов. Система помогает организациям запускать обучение, управлять пользователями и курсами, отслеживать прохождение уроков и анализировать эффективность обучения.

## Ключевая метрика
Годовой регулярный доход с учетом долгосрочных контрактов с организациями.

ARR = количество организаций × средний тариф подписки × процент продлений контрактов

## Состав репозитория
- docs/ — документация, постановка задачи и метрики;
- src/ — исходные файлы и прототипы;
- data/sample/ — демонстрационные данные;
- experiments/ — журнал проверок и гипотез;
- assets/ — схемы и изображения;
- sql/ — структура данных;
- tests/ — проверки проекта.
