# Метрики продукта

## Основная метрика
Годовой регулярный доход с учетом долгосрочных контрактов с организациями.

## Декомпозиция
ARR = количество организаций × средний тариф подписки × процент продлений контрактов

## Метрики
- Количество активных пользователей — главная панель / аналитика.
- Количество зарегистрированных организаций — CRM / админ-панель.
- Количество новых пользователей в месяц — модуль регистрации.
- Среднее время сессии — логи активности.
- Количество пройденных курсов — модуль курсов.
- Частота входов в систему — система авторизации.
- Конверсия регистрации — страница регистрации.
- Конверсия в покупку подписки — платежная система.
- Конверсия завершения курса — модуль обучения.
- Retention 30 — аналитика пользователей.
- Количество продлений подписки — биллинг.
- Доля активных организаций — CRM.
- Количество ошибок авторизации — система логирования.
- Время загрузки страниц — backend / сервер.
- Процент незавершенных курсов — модуль курсов.

## Гипотеза
Если добавить систему автоматических уведомлений о новых уроках и персональные рекомендации курсов, то увеличатся процент завершения курсов и удержание пользователей.
